搭建个人博客的初衷/原因

因为新冠肺炎疫情，今年的寒假已经休了太长时间，但是也给了我一个安静的，独立思考的时间和空间。

最初只是想整理一下自己在各个软件和平台的学习笔记。然而，在连续整理了好几天之后，我崩溃地发现，我花了好几天的结果是，整理的效果和没整理时是一样的！

因为我用了太多的知识管理软件，有的软件和平台确实还不错，然而由于自己的不善管理，让我的学习和管理体系太分散了，自己实在铺得太乱了，这使我再也无法忍受，也自觉这样非常不利于自己的工作。因此我决定找一个所有的东西都能完完全全由自己掌控的平台/空间，多方探索，我最终锁定了——个人博客网站。

事实上，我在搭建个人博客之前，已经为实现这一想法有过多次尝试和试探，比如个人微信公众号，但遗憾的是，我并没有长久经营下去。

搭建博客是为了更好地进行知识管理和总结，是专注写作，也是想知识共享。

添加主页（Home）页面

在_data/navigation.yml文件中header添加如下代码，即可。

  - titles:
      # @start locale config
      en      : &EN       Home
      en-GB   : *EN
      en-US   : *EN
      en-CA   : *EN
      en-AU   : *EN
      zh-Hans : &ZH_HANS  主页
      zh      : *ZH_HANS
      zh-CN   : *ZH_HANS
      zh-SG   : *ZH_HANS
      zh-Hant : &ZH_HANT  關於
      zh-TW   : *ZH_HANT
      zh-HK   : *ZH_HANT
      ko      : &KO       소개
      ko-KR   : *KO
      fr      : &FR       À propos
      fr-BE   : *FR
      fr-CA   : *FR
      fr-CH   : *FR
      fr-FR   : *FR
      fr-LU   : *FR
      # @end locale config
    url: /index.html

在个人博客文章下面添加分享链接

  • 注册AddThis：https://www.addthis.com/
  • 将想要设置分享的应用或链接添加到Inline

  • 在根目录文件_config.yml中对应位置provider改成addthis，并在id处添加AddThis pubid，即可。

    ## => Sharing
##############################
sharing:
  provider: addthis # false (default), "addtoany", "addthis", "custom"

  ## AddThis
  addthis:
    id: # AddThis pubid, e.g. ra-5xxxxxxxxxxx

个人博客访问统计量的显示

  • 工具：使用不蒜子-极简网页计数器

  • 设置步骤：参考网站：不如

  • 本博客网站（以jekyll-TeXt-theme-master为主题）的具体操作如下：

    • 网站总访问量统计设置：

      在根目录下的_includes文件夹下，找到footer.html文件，打开并在紧挨copyright脚注的地方© TouchUMind 2020, Powered by Jekyll & TeXt Theme.的前/后位置，添加如下代码：

      <script async src="//busuanzi.ibruce.info/busuanzi/2.3/busuanzi.pure.mini.js"></script>
            <span id="busuanzi_container_site_pv" style='display:none'>本站总访问量&nbsp;<span id="busuanzi_value_site_pv"></span>&nbsp;次</span>

      即可在博客网站的脚注处看到增加了本站总访问量 XX 次，变成：© TouchUMind 2020, Powered by Jekyll & TeXt Theme. 本站总访问量 XX 次。

    • 单篇文章访问量统计设置：

个人博客评论系统

最初我想添加评论系统可以方便与其他人相互交流与学习，因为我并不能保证自己当前的认知是最正确的，比如我分享了一个我认为最简单高效的工具，但实际上可能有更简单高效的工具，这时其他人可以在评论里提出来，这样大家就可以共同进步了。

但在google了多个个人博客评论插件的推荐之后，并未找到一个完全令我满意的个人博客评论系统。

再者，我的理解，个人博客更倾向于一个私人领域，也许不适合去打扰和被打扰。

因此，我最终决定放弃添加个人博客评论系统。（不过，也许过会我的想法就变了也不一定 。。
